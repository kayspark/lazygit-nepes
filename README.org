* Installation

1. Clone this repo
2. Copy theme YAML to =~/.config/lazygit/=
3. Set =LG_CONFIG_FILE= to the theme path

* Configuration

#+begin_src shell
LG_CONFIG_FILE=~/.config/lazygit/nepes-dark.yml lazygit
#+end_src
